## Cách chỉnh sửa

### Thêm hình ảnh
1. Mở file Word
2. Tìm placeholder `[Hình X.X. ... - Placeholder]`
3. Xóa text placeholder
4. Insert → Pictures → chọn ảnh
5. Thêm caption: References → Insert Caption

### Thêm thông tin sinh viên
1. Mở trang bìa
2. Thay thế `[Họ và tên]`, `[Mã số sinh viên]`, `[Mã lớp]`, `[Tên giảng viên]`

### Cập nhật số trang
1. Sau khi hoàn thiện nội dung
2. References → Table of Contents → Update Table
3. Chọn "Update entire table"
